Research Gap

Many researchers have explored the possibility of using Machine Learning to overcome the challenges of understanding and processing large volumes of documents in human languages to accomplish very specific goals. There have been studies on keyword extraction, key phrase extraction, document clustering and text summarization. However, there has not been a strong focus on the application of such techniques on news which is a unique data set and the value to be gained by doing so. Google News and Monok.com are the prominent existing tools that use machine learning to process news. However, none of them are open source and the underlying technology has not been disclosed. The set of features offered is also very limited and none of them work with Sri Lankan news services.
